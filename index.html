Twitch selfbot

7TV Commands:

Command Name Description
7sa Displays how long a user has been subscribed to a channel using 7TV.
7tv Shows the leaderboard of most used 7TV emotes in the channel.
7tve Shows how many channels a user is an editor for using 7TV.
7u Displays user-specific information on 7TV.
7update Updates a user's preferences on 7TV.

User Information Commands:

Command Name Description
badges Displays all Twitch badges a user has.
color Displays the chat color of a Twitch user.
dn Updates your display name on Twitch.
e Updates your Twitch emote set.
fa Shows how long a user has been following a specific channel.
nh Displays the username history of a Twitch user.
u Displays detailed information about a Twitch user.
song Displays the currently playing song of a Twitch user.

Channel Information Commands:

Command Name Description
chatstats Shows statistics about chat activity in the channel.
chatters Displays a list of all current chatters in the channel.
clear Clears the chat history in the current channel.
ff Displays when a channel received its first follow.
fl Shows the first message a user sent in the channel.
followers Displays the number of followers a Twitch user has.
follows Shows how many channels a Twitch user is following.
founders Lists the founders of the channel.
isbanned Checks if a user is banned in the channel.
mod Lists all channels where the bot owner is a moderator.
mods Lists all moderators in the channel.
pin Pins a specific message in the chat.
team Displays information about a Twitch team.
vc Checks if a user is a VIP in the channel.
vips Lists all VIPs in the channel.

Utility Commands:

Command Name Description
massping Pings everyone in the channel's chat.
mc Checks if a user is a moderator in the channel.
pyr Sends a pyramid message.
r Reloads all bot commands.
rl Sends a random message in the chat.
rs Restarts the bot.
sd Shuts down the bot.
spam Spams a message in the chat.
split Splits a long message into multiple chat messages.
a Spam announces a message in the chat.
wl Adds or removes access to the bot for a specified user.
